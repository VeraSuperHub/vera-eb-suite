# Step 04 — Additional CFA Tests

Add the measurement-model diagnostics that the testing workflow does not cover:

1. Composite reliability / omega
2. Average variance extracted (AVE)
3. Factor correlation matrix
4. HTMT or Fornell-Larcker style discriminant-validity checks when multiple factors exist
5. Modification indices, but only with theory-respecting commentary
6. Optional residual correlation or standardized residual review

Every added diagnostic needs a prose interpretation, not just a table dump.
