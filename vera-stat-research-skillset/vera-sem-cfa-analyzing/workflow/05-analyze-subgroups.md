# Step 05 — CFA Subgroups / Invariance

If a grouping variable is available:

1. Fit configural invariance
2. Fit metric invariance
3. Fit scalar invariance
4. Optionally fit strict invariance only if substantively needed
5. Compare models using ΔCFI / ΔRMSEA / ΔSRMR rather than chi-square only

If invariance fails:
- identify the likely source
- consider partial invariance only with transparent justification
- do not claim latent mean comparability without at least partial scalar support
